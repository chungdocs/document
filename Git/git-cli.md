# Git
> **description**<br>
> 컴퓨터 파일의 변경 사항을 추적하고 여러 명의 사용자들 간에 해당 파일들의  작업을 조율하기 위한 스냅샷 스트림 기반의 분산 버전 관리 시스템

## git init
> 저장소 초기화<br>

```shell
# 현재 폴더(디렉토리)에 git 초기화(버전관리 시작)
git init . 

# 폴더(디렉토리) 생성 후 git 초기화(버전관리 시작)
git init <folder name>
```

### git clone
> 저장소 복제 

```
git clone <repo>
```
저장소 복제

```
git clone <repo> <driectory>
```
